# Security Policy

## Supported Versions

Currently, only the latest version of PyBench is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, please report them by reaching out to the maintainers via Direct Message on the social links provided on the GitHub profile (`@dxnz-id`), or by using GitHub's **Private Vulnerability Reporting** feature if it is enabled on this repository.

We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress in resolving it.
